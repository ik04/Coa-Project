# Install:

(install wrt root folder)

## For client

```bash
cd client/web/quantathon
npm i
```

## For server

```bash
cd server
pip install -r requirements.txt
```

# Run:

## For client
```bash
cd client/web/quantathon
npm run dev
```

## For Server 
```bash
cd server
python manage.py runserver
```
